# Passguard : Evaluateur et générateur de mot de passe.

C'est un programme qui permet de :



*  Evaluer un mot de passe (avec score/message)  

* Générer un mot de passe sécurisé (avec longueur personnalisable)



Il est fait sur python version 3.13.14. 

Pour tester :

``` 

python passguard.py --check "motdepasse" 

python passguard.py --generate [longueur]

```
(expérimental) estimation d'entropie du mot de passe.
